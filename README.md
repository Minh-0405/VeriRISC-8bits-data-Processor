The 8-bit multi cycles processor. The processor is designed to implement custom 8-bit isa in the requirement. However, the memory is divided into inst-mem and data-mem 
instead of only one mem.

This project require deal with FSM machine to create correct control signals, design basic ALU, deal with uncondition jump or skip and basic access with memory.
This project is a stepping stone to gain knowledge about computer architecture and get ready with bigger architecture.
